ftw.deploy
==========

Development
-----------

In order to develop ``ftw.deploy``, you need to install
`pipenv <https://pipenv.readthedocs.io>`_ and follow these instructions:

.. code::

  $ git clone git@github.com:4teamwork/ftw.deploy.git
  $ cd ftw.deploy
  $ pipenv install
